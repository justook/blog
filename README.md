# just.ok的个人博客

这是一个基于 [Hexo](https://hexo.io/) 静态博客框架搭建的个人技术博客，使用 [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) 主题。

## 博客信息

- 博客名称：just.ok的个人博客
- 作者：just.ok
- 语言：中文 (zh-CN)
- 主题：Butterfly v5.2.2
- 博客地址：http://blog.just-ok.xyz

## 技术栈

- 静态博客框架：[Hexo](https://hexo.io/)
- 主题：[Butterfly](https://github.com/jerryc127/hexo-theme-butterfly)
- 部署方式：Git部署
- 永久链接：采用abbrlink插件生成的唯一链接

## 主要功能

- 文章分类管理（categories）
- 标签云（tags）
- 时间轴归档（archives）
- 代码高亮显示
- 响应式设计，支持多端浏览
- 图片画廊（Gallery）
- 友情链接页面
- Google搜索引擎提交支持

## 博客内容

博客主要涵盖以下技术领域：

- 信息安全基础与密码学
- 操作系统相关实践
- 软件逆向工程
- Python编程技巧
- 数据结构与算法
- 网络空间安全数学基础
- 实用脚本工具分享
